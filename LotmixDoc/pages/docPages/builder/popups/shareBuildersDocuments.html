User stories
  • As a Builders manager I want to manage(add new/ delete existed) files that will be available for corresponding client
  • As a Builders manager I want to see views quantity for each file
  • As a Land Builders manager I want to have differentiation of files that I upload by content type(Quote, Drawings, Brochures, Other)
  • As an app owner I want to have white list for upload files format (PDF, JPEG, PNG, JPG)
  • As a Land Builders Manager I want to download already added documents
  • As a Land Builders Manager I want to abort changes if I don't click on submit button
  • As a Land Builders Manager I want to close pop up without any changes apply
