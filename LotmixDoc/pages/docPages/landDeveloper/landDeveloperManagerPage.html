User stories
  • As a Land Developers manager I want to invite new client that will be assign to me as its manager
  • As a Land Developers I want to invite already invited clients by another builders or land developers manager
  • As a Land Developers I want to see in table all of the clients that I invited
  • As a Land Developers I want to edit clients info before it register
  • As a Land Developers I want to remove client from the DB
  • As a Land Developers I want to manage clients short lists
  • As a Land Developers I want to upload documents for clients
